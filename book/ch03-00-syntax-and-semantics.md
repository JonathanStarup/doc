# Syntax and Semantics

The Flix syntax is inspired by Scala, OCaml, and Datalog.

## Syntax Basics

| Primitive                            | Flix          
|--------------------------------------|--------------------------------------|
| Floats                               | `3.1415`                             |
| Integers                             | `3`                                  |
| Strings                              | `"foo bar"`                          |
